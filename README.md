# VRPSPD

El problema de ruteo de vehículos con entregas y recolecciones simultáneas (VRPSPD, por sus siglas en inglés Vehicle Routing Problem with Simultaneous Pickups and Deliveries) es una variante del clásico VRP en la que cada cliente debe ser atendido en una única visita donde el vehículo realiza tanto la entrega como la recolección de mercancía de forma simultánea. El objetivo es determinar un conjunto de rutas de mínimo costo que partan y terminen en un depósito central, satisfagan la demanda de entrega y recolección de cada cliente y respeten la capacidad de carga de los vehículos. Este problema tiene aplicaciones directas en logística de última milla, distribución de productos perecederos y gestión de cadenas de suministro inversas. Al igual que el VRP clásico, el VRPSPD pertenece a la clase de problemas NP-hard, lo que hace inviable su resolución exacta para instancias grandes y motiva el uso de metaheurísticas y heurísticas especializadas.
